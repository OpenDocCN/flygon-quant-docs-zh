## 下载

### Docker

```
docker pull apachecn0/opendoccn-quant-docs-zh
docker run -tid -p <port>:80 apachecn0/opendoccn-quant-docs-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g opendoccn-quant-docs-zh
opendoccn-quant-docs-zh <port>
# 访问 http://localhost:{port} 查看文档
```
