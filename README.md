## 下载

### Docker

```
docker pull apachecn0/flygon-quant-docs-zh
docker run -tid -p <port>:80 apachecn0/flygon-quant-docs-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flygon-quant-docs-zh
flygon-quant-docs-zh <port>
# 访问 http://localhost:{port} 查看文档
```
